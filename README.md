The file: [Milling_Crypt-Villi_Stamp](https://github.com/MHagbok/CAD/blob/b0d58746984285ba20f1fc324be670571fcff70b/Milling_Crypt-Villi_Stamp.f3z), is a CAD design with manufacturing instructions for CNC micro-milling made in AutoDesk Fusion 360. The design was used to create a stamp which mimics the crypt-villi structure of the small intestine for hydrogel structuring. It was used in a bachelor project (Maya Hagbok. Villi-Crypt Integration of Small Intestinal Organ-on-chip Model. 2026) aimed at replicating the villi-crypt structure of the small intestine for an organ-on-a-chip model.
