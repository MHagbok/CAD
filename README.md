The file: https://github.com/MHagbok/CAD/blob/main/Crypt-Villi_Milling.f3z, is a CAD design with manufacturing instructions for CNC micro-milling made in AutoDesk Fusion 360. The design was used to create a stamp for hydrogel structuring in a bachelor project (Maya Hagbok. Villi-Crypt Integration of Small Intestinal Organ-on-chip Model. 2026) aimed at replicating the villi-crypt structure of the small intestine for an organ-on-a-chip model.
